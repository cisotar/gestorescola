# Gestor de Substituições — React

Versão React do Gestor de Substituições, migrada a partir da versão original em JavaScript vanilla.

- React + Vite
- Firebase (Firestore + Hosting + Authentication)

```bash

```

## Desenvolvimento

```bash
npm run dev
```

## Deploy

```bash
npm run build

```
